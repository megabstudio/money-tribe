One-line: Rounded white surface with soft forest-tinted shadow; the base container for content blocks.

```jsx
<Card>
  <CardHeader title="House deposit" subtitle="£4,200 of £10,000" action={<Badge>On track</Badge>} />
  <p>Saving £150 / week.</p>
</Card>
<Card interactive selected>Tap me</Card>
```
`padding` `none|sm|md|lg`. `interactive` adds hover lift; `selected` highlights the border green.
