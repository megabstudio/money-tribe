Read the README.md file within this skill, and explore the other available files.
If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.
If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick map
- `readme.md` — brand voice, visual foundations, iconography (read this first).
- `IMPLEMENTATION.md` — how to install these tokens/components into an existing React project.
- `styles.css` → `tokens/*` — the full token system (colors incl. dark theme, type, spacing, radius).
- `components/*` — Button, IconButton, Input, Textarea, Label, Checkbox, RadioGroup/Radio, Switch, Select, Tabs, Dialog, Badge, Card, Avatar.
- `guidelines/*.card.html` — copy-pasteable specimens.
- `ui_kits/money-tribe-app/` — a full interactive product recreation to learn the patterns from.
- `assets/logo-mark.svg` — the logo.

## Brand in one breath
Community savings. One confident green (vivid lime → deep forest), Manrope (Bold UI) + Bricolage Grotesque (display), soft 12–16px radii, forest-tinted shadows, Radix-style focus rings, light **and** dark themes. Warm plural voice ("Save together"), sentence case, no emoji, £ money.
